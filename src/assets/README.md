# DopaLive Assets

Place your assets here:

- `dopalive_logo.png` - Logo image
- `dopalive_hero_background.png` - Hero background image

If you have Satoshi font files, create a `fonts` subdirectory and place them there:
- `fonts/Satoshi-Regular.woff2`
- `fonts/Satoshi-Medium.woff2`
- `fonts/Satoshi-Bold.woff2`
