# DopaLive - ADHD-Friendly Web Application

## Project Setup

This project is built with Vite, React, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Add your assets to `src/assets/`:
   - `dopalive_logo.png`
   - `dopalive_hero_background.png`

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Brand Guide

### Colors
- Primary: Deep Night Blue (`#1A203A`), Soaring Lavender (`#C4B5FD`)
- Accent: Sweet Coral (`#FF7F50`), Sand Beige (`#F4E9CD`)
- Neutrals: Off-White (`#F8F7FF`), Medium Gray (`#A0AEC0`), Dark Gray (`#2D3748`)

### Typography
- Headings: Satoshi font family
- Body: Inter font family

### Mood
Ethereal, serene, focused, calming
